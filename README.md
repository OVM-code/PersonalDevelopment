# Personal Development

A collection of prompts and tools for self-directed learning and personal growth.

## Structure

- `prompts/` — reusable prompts for learning, planning, and reflection workflows.
  - `book-to-course-prompt.md` — turns a book into a personalized course.
  - `biography-to-course-prompt.md` — turns a real person's life/biography into a
    personalized course (their decisions, turning points, and lessons).
  - Both scale course length and depth to a stated total time budget (a few hours
    vs. a thorough deep-dive), not just a fixed module count.
- `app/` — **CourseCraft**, a web chat app that turns any book into a personalized
  learning journey using the Claude API. See [`app/README.md`](app/README.md) for setup.
